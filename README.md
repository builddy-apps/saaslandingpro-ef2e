# ApexLanding

A premium, high-converting SaaS landing page with animated hero, feature cards, testimonials, pricing tiers, and newsletter signup.

Built with [Builddy](https://builddy.dev) — AI-powered app builder using GLM 5.1.

## Features

- animated_hero_gradient
- sticky_navigation
- smooth_scroll_to_section
- scroll_triggered_animations
- feature_cards_with_icons
- testimonial_carousel
- animated_stats_counter
- three_tier_pricing
- newsletter_signup_with_validation
- dark_mode_toggle
- mobile_responsive
- back_to_top_button
- parallax_effects
- hover_animations
- analytics_tracking

## Quick Start

### Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

### Docker

```bash
docker compose up
```

### Deploy to Railway/Render

1. Push this directory to a GitHub repo
2. Connect to Railway or Render
3. It auto-detects the Dockerfile
4. Done!

## Tech Stack

- **Frontend**: HTML/CSS/JS + Tailwind CSS
- **Backend**: Express.js
- **Database**: SQLite
- **Deployment**: Docker
